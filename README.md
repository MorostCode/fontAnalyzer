# fontAnalyzer开发日志



## 预想界面及功能

1. 双击标签上传字体，再次双击更换字体
2. 界面重置按钮
3. 下拉选框选择字体应用平台和语言
4. 根据平台和语言显示对应的字体详情信息
   - 缺失字段做好相应提示
5. 预计详情信息包括但不限于以下字段
   - 字体在操作系统中的文件名
   - 字体内部文件名
   - 字体大小
   - 字体所含字符字形数
   - 字体创建和最新修改时间
   - 字体版权公告
   - 字体家族名和子家族名
   - 字体唯一标识符
   - 字体版本
   - 字体PostScript名称
   - 商标
   - 制造商
   - 设计师
   - 字体描述
   - 字体供应商URL
   - 设计师URL
   - 许可证
   - 许可证信息URL
   - 更多详细信息……
